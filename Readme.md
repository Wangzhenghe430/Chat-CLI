本项目基于硅基流动（SiliconFlow）大模型 API 开发，实现了一个可在命令行中与 AI 交互的聊天工具。
通过 Git 进行版本控制，并使用 GitHub 完成协作开发。


- 与 AI 进行多轮对话
- 支持切换不同大模型
- 保存对话历史，具备上下文记忆
- 支持读取本地文件内容
- 使用 GitHub 进行版本管理与协作


- Python 3.8+
- 硅基流动 API Key


1. 安装依赖
pip install -r requirements.txt

2. 配置 .env 文件，填写 API Key

3. 启动程序
python main.py chat


使用 Git 进行本地版本管理，上传至 GitHub 仓库，通过分支、PR 完成协作流程。
